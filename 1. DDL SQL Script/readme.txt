To create a schema, please follow steps for setting up the databse.

1) Run schema.sql file to create all the table structure of this schema.

2) For inserting all the dump data, use the SQL scripts inside data folder

3) Run the SQL scripts inside the data folder only in the following order:

    - customer_data.sql 
    - customer_address.sql 
    - vendor_data.sql
    - article_category.sql
    - article_data.sql
    - order_header.sql

4) You an use the ALTER scripts in the outside folder
